# 🎓 Student Information Portal

Welcome to the **Student Information Portal** project!
This project is developed as part of a **Full Stack Development (FSD) project** to manage and display student information in an organized way.

The portal provides a simple interface where users can **register, log in, and manage student details** easily.

👉 **Live Here:** [Open Student Information Portal](https://poornambika.github.io/FSD-project-/) 🌐
